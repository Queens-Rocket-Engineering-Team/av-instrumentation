# Sensors Firmware

This directory contains the firmware for the Sensors Module. This code handles high-frequency sampling and CAN packet transmission for scientific data logging.

## Development Environment
- **IDE:** Arduino IDE
- **Core:** STM32Duino (v2.7.1)
- **Flashing:** SWD via STLinkV2
