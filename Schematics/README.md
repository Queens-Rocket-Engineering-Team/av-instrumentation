# Schematics

This directory contains the schematics for the Sensors Module used in the Avionics system stack.

## Sensors Module
- Interfaces with high-precision analog sensors (Strain Gauges, PTs)
- Uses a 24-bit ADS1256 ADC for high-resolution data
- Communicates data back to the stack via CANBUS
- Includes a NAU7802 bridge digitizer
